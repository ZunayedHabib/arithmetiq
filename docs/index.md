## How This Site Works

Every single question on ArithmetiQ follows the same structure — no surprises:

| What you'll see | What it means |
|---|---|
| 🟢 🟡 🔴 badge | Difficulty level — Easy, Medium, or Hard |
| **STRATEGY** box | The plan before we touch a single number |
| Numbered steps | Each step named, not just algebra floating in space |
| **NOTE** boxes | Only at genuinely tricky moments |
| **FINAL ANSWER** box | Clean summary at the end every time |
